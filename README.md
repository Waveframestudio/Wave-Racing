# Wave Racing

Juego de carreras arcade en 3D construido con `Three.js`, ahora preparado para ejecutarse dentro de un entorno `React + Vite`.

## Características

- Vista en tercera persona con estilo arcade.
- IA de oponentes y sistema de vueltas.
- Efectos visuales (humo, polvo, marcas de neumáticos y líneas de velocidad).
- Sonido de motor/música y minimapa.
- Modo desarrollo con recarga rápida mediante Vite.


## Controles

- `W` / `↑`: acelerar
- `S` / `↓`: frenar / reversa
- `A` / `D` o `←` / `→`: girar
- `SPACE`: derrape / freno de mano
- `R`: reiniciar carrera
